Please follow these steps to install the camera under Ubuntu Linux.
The install procedure has been tested under Ubuntu desktop 12.04.
It should also work for later versions.

1. Argos3D - P100 not connected
2. copy the .rules file by using the following command:
   sudo cp 10-pmd-ubuntu.rules /lib/udev/rules.d/
   (If requested, please enter your password)
3. Connect your Argos3D - P100 and start your demo application.
